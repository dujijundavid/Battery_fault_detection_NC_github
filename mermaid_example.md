# Mermaid → ASCII 转换示例

## 原始 Mermaid

```mermaid
graph LR
    A[输入 x₁, x₂, x₃...] --> B[权重 w₁, w₂, w₃...]
    B --> C[求和 Σw·x]
    C --> D[偏置 b]
    D --> E[激活函数 f]
    E --> F[输出 y]

    style A fill:#e3f2fd,stroke:#1565c0
    style F fill:#4ecdc4,stroke:#0ca678
```

## 转换后 ASCII

```
    输入 x → 权重 w → 求和 Σw·x → 偏置 b → 激活函数 f → 输出 y
    ────────────────────────────────────────────────────────────────────┐
             │ 输入 x₁, x₂, x₃...                 │                │
             │     ↓                                │                │
             └─────────────────────────────────────────────────────────────┘                │
             ┌───────────────────────────────────────────────────────────────────┐
             │              权重 w₁, w₂, w₃...            │                │
             │                ↓                            │                │
             │           求和 Σw·x                      │                │
             │                ↓                            │                │
             │              偏置 b                       │                │
             │                ↓                            │                │
             │              激活函数 f                   │                │
             │                ↓                            │                │
             └───────────────────────────────────────────────────────────────┘                │
             ┌───────────────────────────────────────────────────────────┐
             │              输出 y                        │                │
             └───────────────────────────────────────────────────────────────┘
```

## 替换说明

- 使用 `┌─┐` 和 `└─┘` 绘制方框
- 使用 `│` 和 `─` 绘制箭头
- 可以在 VS Code 中正常显示
